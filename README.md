# Anemia Detection using MobileNetV2

A deep learning-based approach for anemia detection using nail and conjunctival images. The model utilizes MobileNetV2 and transfer learning for accurate and efficient classification.

## Dataset

Due to the large size of the dataset, it is not included in this repository.

Dataset Link:
https://www.kaggle.com/datasets/udayranjankumar/nail-and-conjuctival-images-for-anemia-detection

## Features

- Deep learning-based anemia detection
- MobileNetV2 transfer learning
- Image preprocessing and classification
- Streamlit web application

## Technologies Used

- Python
- TensorFlow
- Keras
- Streamlit
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
